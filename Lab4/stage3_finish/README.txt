~Руководство по работе с блокнотом~

I) INFO:
(Блокнот сохраняет все данные даже после закрытия приложения)
!!("Защита от дурака" не предусмотрена. Пожалуйста, работайте с блокнотом
 согласно нижесказанным указаниям)!!
---------------------------------------------------------------
Num - Хранит упорядоченность заметок (сверху в низ от 1 до ...)
Task - Хранит название заметки (напр. Проснуться)
Time - Хранит задуманное время заметки
Date - Хранит задуманную дату заметки
Status - Хранит степень важности заметки
Result - Хранит степень выполнения заметки
---------------------------------------------------------------

II) Пример блокнота:
-----------------------------------------------------------------------------------
1. Пустой блокнот:
Num Task                           Time   Date         Status          Result
|   |                              |      |            |               |

2. Заполненный блокнот:
Num Task                           Time   Date         Status          Result
|   |                              |      |            |               |
1   Поехать на тусу                11.00  29.09.2022   (Не важно)      Выполнено
2   Пробудиться                    11.45  29.09.2022   (Важно)         Выполнено
3   Поесть                         12.00  29.09.2022   (Важно)         Не выполнено
-----------------------------------------------------------------------------------

III) Команды для работы с блокнотом (!!!):

1. Добавить
  1) Необходимо ввести <Добавить>
  2) Затем программа потребует ввести новую заметку:
      (Пример): Пробудиться, 10.00, 29.09.2022, Важно, Не выполнено
      (! Заметку необходимо указать в подобной форме !).
  3) Далее программа потребует указать номер новой заметки.
      (Необходимо указать число, отличное от 0 и не превосходящее
       номер последней позиции больше чем на 1).
    P.S. Укажите любое место, заметка займёт именно его,
         Отодвинув остальные места.

2. Изменить
  1) Необходимо ввести <Изменить> <номер редактируемой строки>
      Пример: Изменить 2
  2) Затем программа попросит указать изменяемый столбец, новый параметр.
      Пример: Task, Погулять в парке
  P.S. Можно также изменять Num, в таком случае изменённая заметка встанет
       на заданное место, отодвинув прочие.

3. Удалить
  1) Необходимо ввести <Удалить> <номер удаляемой строки>
      Пример: Удалить 4
    Программа Удалит указанную заметку.

4. Закрыть
     Необходимо ввести <Закрыть>
     Приложение выключится, сохранив при этом все данные

5. Очистить
     Необходимо ввести <Очистить>
     Удаляет все заметки в блокноте
